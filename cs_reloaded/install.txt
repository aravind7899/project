To install this module:
First,go to the python Lib/site-packages 
cd Python<version>/Lib/site-packages
for example,if the python version is 2.7
cd Python27/Lib.site-packages
then
Download the module by the command:

git clone https://github.com/aravind7899/cs_reloaded

In order to verify the installation:
In python shell,type

from cs_reloaded import CsOps
